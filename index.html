Como Instalar o App no Celular

📱 INSTALAÇÃO RÁPIDA

Este sistema detecta automaticamente se você está no celular e mostra um botão para instalar. Se não aparecer, siga os passos abaixo:

📱 Para ANDROID (Samsung, Xiaomi, Motorola, etc.):

  1. Abra o site no Google Chrome
  2. Toque nos 3 pontinhos (menu)
  3. Selecione "Instalar app" ou "Adicionar à tela inicial"
  4. Confirme e pronto! 📲

🍎 Para iPHONE (iOS):

  1. Abra o site no Safari
  2. Toque no botão de Compartilhar (quadrado com seta)
  3. Role para baixo e selecione "Adicionar à Tela de Início"
  4. Toque em "Adicionar" e pronto! 📲

Vantagens: Funciona offline, abre como app nativo, notificações ativas
